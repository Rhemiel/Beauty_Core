# Beauty_Core
A web-based management system for Andrea’s Aesthetic and Wellness Clinic in Daet, Philippines. This platform automates scheduling, CRM, inventory, and finances while integrating an AI-driven image generator for nail and hair extension previews. Designed to bridge the digital gap for provincial SMEs using accessible, non-AR technology.


To Push using Git:
when you make new changes in VS Code, you just need to run these three commands to update GitHub:

git add . (Gathers your changes)

git commit -m "Your update message here" (Labels your changes)

git push origin main (Sends them to the web)
